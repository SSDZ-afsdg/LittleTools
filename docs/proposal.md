# 📋 智能记账系统 - 需求文档

> **版本**：v1.0  
> **最后更新**：2026-07-06  
> **状态**：✅ 已确认


## 1. 项目概述

| 项目 | 内容 |
|---|---|
| **项目名称** | 智能记账系统（Smart Accounting） |
| **一句话描述** | 基于本地 AI 的个人记账工具，语音/文字输入，自动分类，月度分析报告 |
| **使用者** | 仅本人（单人使用） |
| **核心价值** | 降低记账门槛 + 让用户真正看懂自己的消费 |


## 2. 功能清单

| 功能 | 状态 | 说明 |
|---|---|---|
| 文字输入记账 | ✅ 必须 | 手动打字录入 |
| 语音输入记账 | ✅ 必须 | 按住说话，语音转文字 |
| 自动归类（AI） | ✅ 必须 | Ollama 本地解析 |
| 月度统计 | ✅ 必须 | 总支出/笔数/日均 |
| 类别分布（饼图） | ✅ 必须 | 可视化展示 |
| 每日趋势（折线图） | ✅ 必须 | 可视化展示 |
| 月度 AI 总结 | ✅ 必须 | AI 生成文字报告 |
| 与上月对比 | ✅ 必须 | 环比 |
| 与去年同期对比 | ❌ 不需要 | — |
| 消费建议（AI） | ✅ 必须 | 基于数据生成建议 |
| 修改/删除记录 | ✅ 必须 | 手机端完整管理 |
| 数据导出（CSV/JSON） | ✅ 必须 | 年度归档时使用 |
| 年度自动归档 | ✅ 必须 | 保留一年，年底自动提醒导出 |
| 预算管理 | ❌ 暂不实现 | 后续可扩展 |
| 多设备同步 | ✅ 必须 | 手机→电脑 手动同步 |


## 3. 使用流程

### 3.1 日常记账
用户说"吃火锅186" → 手机本地暂存 → 继续生活

### 3.2 回家同步
连接Wi-Fi → 手动点击同步 → 数据传到电脑 → AI重新解析 → 生成分析报告

### 3.3 年度归档
满一年 → 系统提醒 → 导出 CSV/JSON → 自动清理旧数据 → 新一年循环


## 4. 非功能性需求

| 需求项 | 要求 |
|---|---|
| **响应速度** | 3 秒内完成（含语音识别+解析） |
| **离线支持** | 手机端必须支持无网络录入，电脑端联网仅用于同步 |
| **数据安全** | 所有数据本地存储，不上传任何云端 |
| **可维护性** | 代码规范，有注释，方便后续扩展 |


## 5. 界面与交互

### 5.1 手机端（PWA）

采用 **4 Tab 布局**：

| Tab | 名称 | 内容 |
|---|---|---|
| ① | 记账 | 语音按钮 + 文字输入框 + 今日汇总 |
| ② | 流水 | 所有记录列表，支持修改/删除 |
| ③ | 分析 | 同步后拉取的月度报告（只读） |
| ④ | 设置 | 同步按钮 + 导出 + 关于 |

### 5.2 电脑端（Jupyter Notebook）

**一页式月度报告**，包含：

- 本月总览卡片（总支出/笔数/日均）
- 类别分布饼图
- 每日趋势折线图
- 与上月对比
- AI 月度总结
- 消费建议


## 6. 技术方案

| 层级 | 选型 | 说明 |
|---|---|---|
| **手机端** | PWA（HTML + CSS + JS） | 可添加到主屏幕，像App一样用 |
| **手机存储** | IndexedDB | 离线暂存数据 |
| **手机解析** | 关键词匹配 | 本地快速分类，不依赖AI |
| **电脑端** | Jupyter Notebook | 数据分析和AI调用 |
| **电脑AI** | Ollama（qwen2.5:7b） | 本地运行，隐私安全 |
| **电脑存储** | MySQL | 主数据库 |
| **同步协议** | HTTP REST API（FastAPI） | 局域网通信 |


## 7. 数据结构

每条记录存储以下字段：

| 字段 | 说明 |
|---|---|
| `id` | 唯一标识 |
| `amount` | 金额 |
| `category` | 类别（餐饮/交通/购物/娱乐/居住/医疗/教育/其他） |
| `description` | 简短描述 |
| `date` | 消费日期 |
| `raw_text` | 原始输入 |
| `created_at` | 录入时间（自动生成） |
| `ai_parsed` | AI解析结果（保留） |
| `status` | 状态（draft/pending/synced） |
| `version` | 版本号，冲突检测用 |
