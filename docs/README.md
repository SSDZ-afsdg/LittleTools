# 项目简介
Recording 是一个语音输入为主，文字输入为辅的账单记录工具。用户可以通过自然指令，系统能自动理解并完成消费的记录与归类，彻底摆脱繁琐的统计。

## 一句话
说话就能记录消费

## 核心流程
>用户打开手机App → 说"吃火锅花了186"
        
1. 手机端：语音转文字 → 关键词匹配（本地快速解析）
        
2. 存到手机本地（IndexedDB），状态标记为"待同步"
        
3. 回家连上Wi-Fi → 点"同步"按钮
        
4. 数据发送到电脑 → Ollama AI重新解析 → 存入SQLite数据库
        
5. 电脑生成月度报告/图表/建议 → 手机拉取结果显示
